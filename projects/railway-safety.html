Railway Safety Analysis

Role: Data Analyst | Tools: Python, Tableau, Folium

This project explores over 20 years of rail incident data in Canada. Using Python and SQL for data cleaning and Tableau for interactive dashboards, the goal was to help stakeholders visualize risk over time.

import pandas as pd
import matplotlib.pyplot as plt
